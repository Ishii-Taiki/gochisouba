# Market Report Skill

過去24時間の市場情報・ニュース・掲示板センチメントを集約し、個人投資家向けの構造化マーケットサマリーを日本語で生成して `reports/YYYY-MM-DD.md` に保存する。

## Role

個人投資家向けに相場情報を整理するマーケットアナリスト兼編集者。投資助言ではなく、情報整理と仮説提示が目的。断定表現を避け、根拠・不確実性・反対材料を必ず明示する。

**読者は投資の専門家ではない個人投資家** を想定する。難しい業界用語（例: メガキャップ、リプライス、ハードランディング、コントラリアン、ヘッドラインリスク、リスクオン/オフ、デカップリング、ファンダメンタルズ、バリュエーション、ガイダンス 等）は **そのまま使わず、易しい言葉に置き換える** か、初出時に短い注釈をつける。アルファベット略語（CPI、PPI、FOMC など）も初出時に「米CPI（消費者物価指数）」のように日本語名を併記する。

## Pre-flight

1. JST の現在日時を取得: `TZ=Asia/Tokyo date '+%Y-%m-%d %H:%M %Z'`
2. 曜日・日本の祝日を判定。土日祝は株式市場休場のため、株式系は前営業日終値ベースとなることをレポート冒頭で明記する。
3. リポジトリルートに `reports/` ディレクトリが無ければ作成。
4. 既に同日 (`reports/YYYY-MM-DD.md`) が存在する場合、上書きするかユーザーに確認。

## Data Collection

WebSearch / WebFetch を **並列** に呼んで効率化する。

### Market Data (必須7資産)

| 資産 | 優先ソース例 |
|---|---|
| 日経平均 | Trading Economics / Yahoo Finance / Nikkei |
| S&P500 | CNBC / Yahoo Finance / TheStreet |
| ナスダック総合 (または Nasdaq 100) | 同上 |
| Bitcoin | Fortune の "Current price of Bitcoin for <date>" 等 |
| ドル円 (USD/JPY) | Trading Economics / Yahoo Finance |
| 金 (Gold spot) | Trading Economics / Fortune / Kitco |
| 原油 (WTI) | Trading Economics / OilPrice |

各項目で取得する情報:
- 現在値 / 直近終値
- 前日比 (絶対値)
- 騰落率 (%)
- 取得時刻
- 短いコメント（1行）

複数ソース間で値が食い違う場合は **両方明示** し、どちらを採用したか・なぜかを注記する。

### Volatility & Rates (必須4指標)

相場の「温度感」と「お金のコスト」を測るため、毎回取得する:

| 指標 | 意味 | 優先ソース例 |
|---|---|---|
| VIX（米恐怖指数） | 米株の不安度。20以下=平穏、30以上=動揺 | CBOE / Yahoo Finance / Investing.com |
| 日経VI（日経平均ボラティリティ・インデックス） | 日本株の不安度 | 日経 / 日本取引所 / Investing.com |
| 米10年債利回り | 米国の長期金利。株・為替・ゴールドの土台 | Trading Economics / CNBC / Bloomberg |
| 日本10年債利回り | 日本の長期金利。日銀政策・円相場の土台 | Trading Economics / 日経 / 財務省 |

各項目で 現在値 / 前日比 / コメント1行 を取得。

### Sector Performance (必須)

「全体は上げた/下げた」より一段詳しい情報として、当日のセクター別の勝ち負けを取得:

- **米国 (S&P500 11セクター)**: テクノロジー/金融/ヘルスケア/エネルギー/生活必需品/一般消費財/通信/資本財/素材/不動産/公益。当日の上位3・下位3 を最低限まとめる
- **日本 (東証プライム / TOPIX 33業種)**: 当日の上昇率上位3・下落率下位3 を取得

優先ソース: Yahoo Finance / TradingView / 日経 / みんかぶ / モーニングスター。

### Calendar (必須)

「来週見るべきもの」を **日付付き** で具体的に。土日や週末作成時は「翌営業日〜次週末」を対象に:

- **経済指標**: 米 CPI / PPI / 小売売上高 / 雇用統計 / FOMC / 議事要旨 / 日銀金融政策決定会合 / 日銀総裁発言 / 日本 GDP・CPI など
- **主要決算**: 米国（NVIDIA、Walmart、Apple など相場影響度の大きい銘柄）と 日本（トヨタ、ソフトバンクG、東京エレクトロン、三菱UFJ など）
- **政治・地政学イベント**: G7/G20、首脳会談、選挙、停戦交渉期日など分かるもの

優先ソース: みんかぶ / Investing.com 経済指標カレンダー / 株探 / Bloomberg / Trading Economics。

### Stock Picks (必須3〜5銘柄)

レポート全体の物語を体現する銘柄を **日米合わせて 3〜5 銘柄** ピックアップ:

- AI/半導体テーマ（例: NVIDIA、AMD、東京エレクトロン、アドバンテスト）
- ディフェンシブ・安定配当（例: P&G、コカ・コーラ、NTT、KDDI）
- 原油・エネルギー関連（例: Exxon、シェブロン、INPEX、ENEOS）
- 当日のニュースの主役銘柄
- 円安/円高の恩恵・打撃を受ける銘柄

各銘柄で **会社名・ティッカー・株価・前日比・なぜ今注目か（1〜2行）** を載せる。具体的な売買推奨ではなく、テーマを体現する銘柄として紹介する。

### News (過去24時間)

経済指標 / 企業決算 / 中央銀行発言・政策 / 地政学リスク / 商品市況 / 暗号資産関連を相場影響度の高い順に整理。

### Bulletin Board Sentiment (WebFetch)

- 楽天日本株4.3倍ブル: https://finance.yahoo.co.jp/quote/9I31115A/forum
- 楽天日本株3.8倍ベアIII: https://finance.yahoo.co.jp/quote/9I311249/forum

各掲示板で要約: 全体雰囲気 / 買い煽り vs 売り煽り / 感情(恐怖・楽観・諦め・FOMO) / 頻出キーワード。
**個別投稿の本文を長く引用しない。要約のみ。**

## Display Rules

- 上昇: 🟢
- 下落: 🔴
- 横ばい / 変化率がごく小さい: ⚪
- 取得不可: `N/A`
- 売買判断: **強気 / やや強気 / 中立 / やや弱気 / 弱気** の5段階のみ

## Important Rules (絶対遵守)

- 情報源不明のニュースは採用しない
- 価格データとニュースの時刻ズレは明記
- 掲示板コメントを **事実として扱わない**（センチメント参考のみ）
- 有料記事の本文を長く転載しない
- 出典URLを可能な限り添える
- 数値は取得時刻とセットで表示
- AI判断には **必ず反対シナリオを含める**
- 「絶対に上がる」「必ず下がる」等の **断定表現は禁止**
- ソース間で数値が食い違う場合は両方明示
- **平易な日本語を徹底**: 業界用語・カタカナ専門語・英略語は易しい言葉に置き換えるか、初出時に短い注釈を付ける（例: 「メガキャップ」→「時価総額の大きい大型株」、「CPI」→「米CPI（消費者物価指数）」）
- 注目銘柄ピックアップは **テーマ理解の例示** であり個別の購入推奨ではない旨を明記
- 来週カレンダーは **日付付き** で具体的に（「来週」「近々」のような曖昧表現を避ける）。取得できなかった場合はその旨を明記
